
#  Responsive Shopping Website

A **simple, fully responsive shopping website** built using **HTML, CSS, and JavaScript**.  
The website includes product cards, smooth navigation, and basic add-to-cart UI effects.  

##  Features

- Responsive layout for mobile, tablet, and desktop screens  
- Clean and modern product card design with hover effects
- Smooth navigation across sections  
- Basic add-to-cart interface behaviour (UI-level only)  
- Built using Flexbox and media queries

##  Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**

##  How to Use

1. Clone or download the repository  
2. Open `index.html` in any web browser  
3. Browse the website and interact with the UI  

_No server setup is required._

##  Possible Future Enhancements

- Search and filter functionality  
- Cart persistence using localStorage  
- Backend integration for real products and orders  
- Login / signup pages  

##  Note

This project is intended as a **frontend UI demonstration** and does not include real checkout or backend services.
